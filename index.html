📋 Как настроить кастомные поля
Шаг 1: В PlanFix перейди: Управление аккаунтом → Задачи → Пользовательские поля
Шаг 2: Создай 4 поля типа «Флажок» (чекбокс): «Как работаем», «Презентация», «Призыв к действию», «Счёт скинул»
Шаг 3: Создай 1 поле типа «Строка» или «Текст»: «Итог звонка»
Шаг 4: Скопируй ID каждого поля (виден в URL при редактировании) и вставь выше
Шаг 5: API-токен: Управление аккаунтом → Доступ к API → REST API → Создать токен
